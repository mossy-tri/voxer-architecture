# User Profile API

User profile management, search, and account settings.

## Endpoints

- `3/cs/public_profiles` - Get user profile(s) by user ID(s)
- `1/cs/username_public_profile` - Get user profile by username
- `2/cs/edit_profile` - Update user profile information
- `2/cs/change_password` - Change user password
- `2/cs/username_for_user_id` - Change username for a user
- `username/available/1` - Check if username is available
- `4/cs/progressive_search` - Search for users by query
- `2/cs/user_delete_account` - Delete user account
