# Timeline & Sync API

Data synchronization and real-time update acknowledgment.

## Endpoints

- `3/cs/timeline` - Get timeline data (messages, profiles, contacts)
- `1/cs/ack_timeline` - Acknowledge timeline with high water mark
- `1/cs/ack_timeline_on_delivery` - Acknowledge timeline on delivery
- `/2/ack_updates` - Acknowledge real-time updates
