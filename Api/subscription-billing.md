# Subscription & Billing API

Subscription lifecycle, payment methods, and billing management.

## Endpoints

- `1/cs/user/subscriptions` - Get or modify managed subscriptions
- `1/cs/user/cancel_subscription` - Cancel subscription
- `1/cs/user/renew_subscription` - Renew subscription
- `1/cs/user/subscription_status` - Get subscription status
- `1/cs/user/payment_method` - Update payment method
- `1/cs/user/billing_history` - Get billing history
- `1/cs/user/apply_coupon` - Apply coupon or promo code
- `1/cs/subscription_plans` - Get available subscription plans
- `1/cs/user/upgrade_subscription` - Upgrade subscription
- `1/cs/user/downgrade_subscription` - Downgrade subscription
- `1/cs/user/usage_stats` - Get subscription usage statistics
- `1/cs/user/auto_renewal` - Set subscription auto-renewal
