# iOS Client

GitHub repository: https://github.com/voxer/ios

## Overview

Native iOS application built with Xcode and CocoaPods (Swift/Objective-C). Published to the App Store as "Voxer Walkie Talkie Messenger" (Bundle ID: com.rebelvox.voxer-lite). Uses fastlane for deployment automation and includes both iOS and watchOS targets (minimum watchOS 8.7+). Recent development includes version updates (currently 3.30.9), message recall endpoint updates (v3), and UI refinements including disabling the Liquid Glass effect.
